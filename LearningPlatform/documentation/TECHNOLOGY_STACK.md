# BrainStack Technology Stack

| Layer | Technology |
|-------|-----------|
| Framework | Next.js 15.5 (App Router) |
| Language | TypeScript 5 |
| CMS | Payload CMS 3.72 |
| ORM | Prisma 7.3 |
| Database | PostgreSQL (target/runtime: 16) |
| Authentication | Auth.js v5 (`next-auth` beta) |
| Object storage | AWS S3 / S3-compatible (with local filesystem fallback) |
| Styling | Tailwind CSS 4 + Radix UI |
| AI providers | Anthropic and OpenAI |
| Testing | Vitest 4 + Testing Library + Playwright |
| Container | Docker (multi-stage) |
